# .clear()

### Overview

Removes all elements within the set.

`clear(): AdvancedSet<T>`

### Parameters

None

### Usage

```js
const setA = new AdvancedSet(1, 2, 3, 4, 5);
setA.clear(); // AdvancedSet()

// Original object is changed, as well as the return value
console.log(setA); // AdvancedSet()
```
