# .has()

### Overview

Returns if the set has the given element.

`has(val: T): boolean`

### Parameters

`val` - The element to look for.

### Usage

```js
const setA = new AdvancedSet(1, 2, 3, 4, 5);
setA.has(1); // true
setB.has(0); // false
```
