# .isEmpty()

### Overview

Returns whether or not the set contains no elements.

`isEmpty(): boolean`

### Parameters

None

### Usage

```js
const setA = new AdvancedSet();
setA.isEmtpy(); // true

const setB = new AdvancedSet(1, 2, 3);
setB.isEmtpy(); // false
```
