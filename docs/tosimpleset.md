# .toSimpleSet()

### Overview

Returns the current `AdvancedSet` as a simple JavaScript `Set`.

`toSimpleSet(): Set<T>`

### Parameters

None

### Usage

```js
const setA = new AdvancedSet(1, 2, 3, 4, 5);
set.toSimpleSet(); // Set([1, 2, 3, 4, 5])
```

### Details

Introduced in: 0.0.1

Last updated in: 0.0.1
